# Hookah Landing

Adaptive landing page for hookah lounge with interactive form, sliders and smooth animations.

## Demo

- **Live site**: https://vmuravlev.github.io/hookah-landing/

## Features

- Fully responsive design
- Contact/order form with validation
- Sliders/carousel for gallery/products
- CSS/JS animations for sections, buttons, hover effects
- Smooth scrolling elements

## Tech Stack

- HTML5
- CSS3 (Grid/Flexbox, Animations, Media Queries)
- Vanilla JavaScript (Sliders, Form logic)

## Quick Start

```bash
# Clone repo
git clone https://github.com/vmuravlev/hookah-landing.git
cd hookah-landing

# Open index.html in browser
# or if using bundler:
npm install  # if package.json present
npm run dev  # or npm start
